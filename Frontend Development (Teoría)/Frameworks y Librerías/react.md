# **Ciclo de Vida en React** üîÑ‚öõÔ∏è

El **ciclo de vida** de un componente en React se refiere a las diferentes etapas por las que pasa un componente desde su **creaci√≥n**, **actualizaci√≥n** y **desmontaje**.

## **1. Antes de los Hooks: Ciclo de Vida en Componentes de Clase** üèõÔ∏è

Antes de los Hooks, los componentes de clase ten√≠an m√©todos espec√≠ficos para manejar su ciclo de vida:

| M√©todo                 | ¬øCu√°ndo se ejecuta?                                                                                    |
| ---------------------- | ------------------------------------------------------------------------------------------------------ |
| `componentDidMount`    | Despu√©s del primer render (equivalente a `useEffect(() => {...}, [])`).                                |
| `componentDidUpdate`   | Cuando el componente se actualiza (equivalente a `useEffect(() => {...}, [dependencias])`).            |
| `componentWillUnmount` | Antes de que el componente se elimine (equivalente a `useEffect(() => {... return () => {...}}, [])`). |

### **Ejemplo en un Componente de Clase**

```jsx

  constructor() {
    super();
    this.state = { contador: 0 };
  }

  componentDidMount() {

  }

  componentDidUpdate(prevProps, prevState) {

  }

    return (
      <div>
        <p>Contador: {this.state.contador}</p>
        <button
          onClick={() => this.setState({ contador: this.state.contador + 1 })}
        >
          Incrementar
        </button>
      </div>
    );
  }
}

export default CicloDeVida;
```

## **2. Despu√©s de los Hooks: `useEffect` en Componentes Funcionales** üèóÔ∏è

üìå **`useEffect` se ejecuta despu√©s de que React actualiza el DOM.**

‚úÖ **Se ejecuta solo una vez, cuando el componente se monta.**

## **3. Limpieza de efectos con `useEffect`: Equivalente a `componentWillUnmount`** üóëÔ∏è

Cuando un componente se desmonta, podemos **limpiar efectos secundarios** como:

- **Eliminar event listeners**
- **Detener intervalos o timeouts**
- **Cancelar suscripciones a APIs**

üìå Para esto, **`useEffect` retorna una funci√≥n de limpieza.**

### **Ejemplo: Intervalo de tiempo con limpieza**

```jsx

      console.log("Temporizador detenido ‚ùå");

```

## **üìå Resumen**

| M√©todo de Clase        | `useEffect` Equivalente                         |
| ---------------------- | ----------------------------------------------- |

| `componentDidUpdate`   | `useEffect(() => {...}, [dependencias])`        |
| `componentWillUnmount` | `useEffect(() => {... return () => {...}}, [])` |

üìå **Antes (Clase)**: Us√°bamos m√©todos como `componentDidMount`.  
üìå **Ahora (Funcional)**: Usamos `useEffect` con dependencias para controlar cu√°ndo se ejecuta.  
üìå **Para limpiar efectos secundarios**, `useEffect` retorna una funci√≥n que se ejecuta cuando el componente se desmonta.
